# ☕ Java Backend Learning Journey

A repository focused on building a solid Java foundation through a practical and domain-oriented approach.

---

## 📖 About the Repository

This repository focuses on mastering core Java concepts, from language fundamentals to database integration and architectural patterns.

### 🎯 Core objectives:

- Structured and progressive learning
- Practice-driven approach
- Clean code and performance focus

---

## 🧩 Project Structure

The project follows a modular and highly isolated structure:

### 🔹 Core Principles

- **Full Isolation**
  - Each lesson/exercise is independent  
  - Has its own `main` method  
  - Runs without dependencies on other modules  

- **Consistent Organization**
  - Directories use `snake_case`
  - Clear separation by modules and topics

## 🚀 How to Run

Each module is self-contained. Simple workflow:

```git clone https://github.com/your-username/your-repository.git```

## ▶️ Execution

- Open the project in your IDE → Recommended: IntelliJ IDEA
- Navigate to the desired module

Run the class containing:
