## Employee Payment System

A company has **in-house employees** and **outsourced employees**.

For each employee, the following data must be recorded:

- Name
- Hours worked
- Hourly rate

Outsourced employees also have an **additional expense**.

### Payment Rules

- **In-house employees**
    - Payment = hourly rate × hours worked

- **Outsourced employees**
    - Payment = hourly rate × hours worked
    - Plus a bonus equal to **110% of the additional expense**

### Program Requirements

- Read the data of **N employees** (where **N is provided by the user**).
- Store all employees in a **list**.
- After reading all the data, display:
    - The **name**
    - The **payment amount**

The output must follow the **same order** in which the employees were entered.
