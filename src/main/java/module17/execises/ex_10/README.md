# Exercise 10 — Integrated Evaluation and Storage System

## Detailed Description

A company wants to:

- Store generic records
- Calculate the maximum record
- Transfer data between structures
- Generate a universal report

The system must be fully reusable across different domains.

---

## Required Classes

- `StorageService<T>`
- `EvaluationService`
- `TransferService`
- `ReportService`

---

## Functional Rules

- All components must be reusable.
- Properly apply type bounds and wildcards.
- Follow generics best practices, including PECS where appropriate.

---

## Technical Objective

- Full integration of generics
- Bounded types
- Wildcards
- PECS principle
