## Detailed Description

Implement a task controller using a `List<String>`.  
The system must allow adding tasks, removing tasks **by name**, removing tasks **by index**, and checking the **total number of tasks**.

## Required Classes

- None besides the main class.

## Attributes

- `tasks` (List<String>)

## Methods

- `addTask()`
- `removeByName()`
- `removeByIndex()`
- `listAll()`

## Rules

- It must not be possible to remove a **nonexistent task**.
- The list must display its **size whenever it is modified**.

## Expected Inputs

- User commands to add and remove tasks.

## Expected Outputs

- Final list of tasks.

## Technical Objective

Practice **List** operations: `add`, `remove`, `get`, and `indexOf`.
