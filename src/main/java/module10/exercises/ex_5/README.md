- These values must be converted to **wrapper objects** and then back to **primitive types**.
- Always display the **type of the converted variable**.

- Integer and double values.

## Expected Outputs

- Clear demonstrations of **boxing and unboxing**.

## Technical Objective

Practice **boxing**, **unboxing**, and the use of **wrapper classes**.
