## Detailed Description

Read a number **N** of students and store their grades in an array of objects.  
Calculate and display the **highest grade**, **lowest grade**, and the **arithmetic average**.

## Required Classes

- **Grade**

## Attributes

- `grade1` (Double)
- `grade2` (Double)

## Methods

- `getValue()`

## Rules

- The array must contain **only valid Grade objects**.
- Average = sum / quantity.
- There must be **at least one student**.

## Expected Inputs

- **N**, the number of students.
- **N values** (grade 1 and grade 2 for each student).

## Expected Outputs

- Highest grade
- Lowest grade
- Average

## Technical Objective

Reinforce the use of **arrays of objects**, input handling, and **array traversal**.
