## Detailed Description

Create a system that receives a list of names and filters **only the names that start with a specific letter**.

- None.

## Attributes

- `names` (List<String>)
- Auxiliary list `filtered`

## Methods

- `filterByInitial(char initial)`

## Rules

- The filtering must be done **manually** by iterating through the list and adding valid items to another list.

## Expected Inputs

- Number of names
- List of names
- Letter to filter by

## Expected Outputs

- A filtered list containing only the names that start with the given letter.

## Technical Objective

Practice **List** usage, **sequential traversal**, and the creation of **auxiliary lists**.
