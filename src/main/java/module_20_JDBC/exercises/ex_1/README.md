## EXERCISE 1 — Simple Product Registration

### Detailed Description

A store needs a basic system to register products in a database. The system must allow adding new products and listing all registered products.

---

### Required Classes

- DB
- DBException
- DBIntegrityException
- Product
- Program (main)

---

### Attributes

#### Product

- `Long id`
- `String name`
- `Double price`

---

### Methods

- insert product
- list products

---

### Operating Rules

Each product must be persisted in the database.

IDs must be automatically generated by the database.

The system must retrieve and display all registered products.

---

### Expected Inputs

- product name
- price

---

### Expected Outputs

- list of products with id, name, and price

---

### Technical Objective

Introduction to JDBC: connection, simple insert, and select.
